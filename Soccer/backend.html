AI Image Generator

About This Project

The image generation can be slow sometimes so give it up to 10 seconds before an image is generated.

This project is a simple AI image generator that takes a text description and generates an image using the Pollinations API. The backend is built in Go, which handles the API requests and manages CORS for frontend access.

The Pollinations API allows users to generate images based on descriptive text. You can find more about their API here.

This project demonstrates how to connect a frontend application with a backend server and make API calls to generate dynamic content based on user input.

The source code for this project is available on my GitHub for those interested in exploring the implementation further.
