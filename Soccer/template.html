Template Pattern in Soccer

What is the Template Pattern?

The Template Pattern is like a recipe you follow step-by-step. Some steps always stay the same, like the bread in a sandwich. But other parts, like the filling, can change.

In programming, this pattern is useful when you want to keep certain steps consistent but allow others to vary based on specific needs.

Soccer Scenario

Consider Ronaldo and Messi. Both follow a similar structure in a game: they warm up, play in their unique style, execute signature moves, and celebrate. While the warm-up and celebration are the same, their play styles and moves differ.

Template Code Example


abstract class SoccerPlayer {

    // Template method defining the skeleton of a soccer player's game
    public final void playGame() {
        warmup();
        setPlayStyle();
        executeSignatureMove();
        celebrate();
    }

    //Both will warm up in the same manner.
    private void warmup() {
        System.out.println("Warming up with some dribbling and stretches.");
    }

    //This is where these two players behaviors will differ.
    protected abstract void setPlayStyle();
    protected abstract void executeSignatureMove();

    //Both will celebrate in the same way.
    private void celebrate() {
        System.out.println("Celebrates the victory!");
    }
}
      

// Concrete class for Ronaldo's play style and signature move
class Ronaldo extends SoccerPlayer {
    @Override
    protected void setPlayStyle() {
        System.out.println("Ronaldo does his step overs.");
    }

    @Override
    protected void executeSignatureMove() {
        System.out.println("Ronaldo performs his signature 'knuckleball' free kick.");
    }
}

// Concrete class for Messi's play style and signature move
class Messi extends SoccerPlayer {
    @Override
    protected void setPlayStyle() {
        System.out.println("Messi does his body feints.");
    }

    @Override
    protected void executeSignatureMove() {
        System.out.println("Messi curls the ball in from the top of the 18-yard box.");
    }
}

Running the Template Pattern


public class SoccerGame {
    public static void main(String[] args) {
        SoccerPlayer ronaldo = new Ronaldo();
        SoccerPlayer messi = new Messi();

        System.out.println("Ronaldo's Game:");
        ronaldo.playGame();

        System.out.println("\nMessi's Game:");
        messi.playGame();
    }
}

Messi does his body feints.
Messi curls the ball in from the top of the 18-yard box.
Celebrates the victory!

Notice that both players follow the same steps, but their play styles and signature moves differ.
