This project is a work in progress, but the Selenium web scraper is functional. It runs in headless mode, so you won't see the Chrome browser pop up while it scrapes the data.

AH, I had to make changes becuase when pushing this to Render, there was no way to trigger the Selenium python without having to do some weird hacky thing, so rewrote this into Go. Honestly, way better. It is way faster than python.
